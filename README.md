# ◈ iOS Auto Clicker Pro (Yalla Ludo Edition) ◈

مشروع متكامل واحترافي لإنشاء تطبيق Auto Clicker لنظام iOS، مصمم خصيصاً ليتم حقنه في الألعاب مثل **Yalla Ludo**. يتميز بواجهة رسومية عائمة (Floating GUI) وتصميم عصري.

## 📸 المميزات (Features)
- **واجهة احترافية**: قائمة تحكم منظمة تحتوي على أزرار التشغيل والإيقاف.
- **سحب وإفلات**: يمكنك تحريك لوحة التحكم في أي مكان على الشاشة أثناء اللعب.
- **سرعة عالية**: تم ضبط المحرك ليعمل بسرعات فائقة تتناسب مع متطلبات الألعاب.
- **بناء تلقائي**: مدمج مع GitHub Actions لبناء المشروع تلقائياً وتحويله إلى ملف جاهز.

## 🚀 كيفية الاستخدام (How to Use)

### 1. الرفع إلى GitHub
قم برفع كافة ملفات هذا المجلد (7 ملفات) إلى مستودع جديد في حسابك على GitHub.

### 2. تحميل الملف الجاهز
- بعد رفع الملفات، انتقل إلى تبويب **Actions** في مستودعك على GitHub.
- ستجد عملية بناء بدأت تلقائياً باسم "Build Tweak".
- بعد اكتمالها (ظهور علامة صح خضراء ✅)، اضغط على اسم العملية.
- انزل لأسفل الصفحة في قسم **Artifacts** ستجد ملفاً باسم **Tweak-Build**، قم بتحميله.

### 3. الحقن والتثبيت
- استخرج ملف الـ `.deb` أو الـ `.dylib` من الملف المحمل.
- استخدم أداة **Sideloadly** على الكمبيوتر لاختيار تطبيق Yalla Ludo وحقن الملف فيه.
- قم بتثبيت التطبيق الناتج على هاتفك واستمتع!
